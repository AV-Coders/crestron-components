# AVCoders.Crestron.DSP

Volume controls bound to Crestron signals. Part of the [AV Coders crestron-components repository](https://github.com/AV-Coders/crestron-components). Targets **.NET 8.0**.

## Install

```bash
dotnet add package AVCoders.Crestron.DSP
```

Available from the AV Coders GitHub Packages feed (`https://nuget.pkg.github.com/AV-Coders/index.json`) and nuget.org. MIT licensed.

## What's inside

- `SigBasedVolume` - a `VolumeControl` over Crestron sigs
- `UcEngineMtrSpeakerVolume` - UC Engine / Teams room speaker volume
